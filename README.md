## Example steup 

defaultFont - set default font size
step - set step
min - set minium font size
max - set max font size
suffix - free to use "%", "px", "em" or "rem"
```
 <TextResizer  defaultFont="100" step="20" min="60" max="180" suffix="%" />
 
```
